# 高考志愿填报·红宝书 V0.1

**从「能录取」走向「适合发展」——位次 × 院校 × 专业 × 城市 × 产业 × 就业**

- 作者：**jason.wa**
- 当前版本：**v0.1**
- 状态：已发布

## 内容描述

本书系统讲解志愿填报的方法论、核查流程与决策工具，帮助考生在分数约束下做出兼顾录取安全与长期发展的选择。涵盖位次分析、录取规则、院校专业选择、城市产业就业关联、大学生涯规划等主题。

## 目录

1. 位次与录取规则
2. 院校选择方法
3. 专业分析框架
4. 城市与产业就业
5. 大学生涯规划
6. 志愿填报决策流程
7. 常见问题与风险防控

## 发布新版本

1. 更新 `VERSION`，更新书稿和 `CHANGELOG.md`
2. 运行本地校验
3. 提交并推送
4. 打 Tag 并推送

```bash
git tag -a 3.高考志愿填报红宝书-v0.1 -m "3.高考志愿填报红宝书: Release v0.1"
git push origin 3.高考志愿填报红宝书-v0.1
```

Tag 推送后 Release 工作流会自动上传产物。
