# MyBook 仓库

一个仓库管理多本书籍。

## 目录结构

```
books/
├── book-1/        # 第一本书
│   ├── book.md
│   ├── preface.md
│   └── chapters/
├── book-2/        # 第二本书
│   ├── book.md
│   ├── preface.md
│   └── chapters/
└── book-3/        # 第三本书
    ├── book.md
    ├── preface.md
    └── chapters/
```

## 使用

每本书独立维护，互不干扰。
